## Creation History

```bash
pnpm create svelte@latest svelte-v5-ex1
cd svelte-v5-ex1
pnpm install
pnpm add -g pnpm
pnpm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm run dev -- --open
pnpm i -D unplugin-icons
pnpm install --save-dev @iconify/svelte
```
